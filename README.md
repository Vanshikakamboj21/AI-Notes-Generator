🚀 ExamNotesAI

ExamNotesAI is a full-stack AI-powered web application that generates structured, exam-ready notes, diagrams, and charts instantly.

It helps students transform topics into clear, revision-focused content using AI for faster and smarter exam preparation 📚⚡

✨ Features

🧠 AI-powered notes generation

📊 Auto-generated diagrams & charts

🔐 Secure JWT authentication

💳 Credit-based usage system

📄 PDF export support

🌐 Fully responsive UI

⚡ Fast and optimized backend
